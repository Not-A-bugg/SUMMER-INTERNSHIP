# 🏨 StayEase - Hotel Booking Website

StayEase is a Hotel Booking Website developed as a university web development project. The goal of this project is to provide a simple and user-friendly platform where users can browse hotels, view details, and make room bookings.

> **Current Status:** Phase 1 (Frontend using HTML & CSS)

---

## 📌 Project Objectives

- Design a modern hotel booking website.
- Provide a clean and responsive user interface.
- Display hotel information in an attractive layout.
- Build the frontend before adding backend functionality.

---

## 🚀 Technologies Used

- HTML5
- CSS3
- Google Fonts
- Font Awesome
